# Andrew Koja - Personal Portfolio Website

A clean, modern, and responsive personal portfolio website built with HTML, CSS, and JavaScript.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern UI**: Clean, professional design with smooth animations
- **Interactive Navigation**: Smooth scrolling and mobile-friendly hamburger menu
- **Contact Form**: Functional contact form with validation
- **Project Showcase**: Grid layout for displaying your projects
- **Skills Section**: Clean display of your technical skills
- **Social Links**: Easy integration with your social media profiles

## Customization Guide

### 1. Profile Picture
Replace `profile.jpg` with your actual profile photo. The image should be:
- Square aspect ratio (1:1)
- At least 300x300 pixels
- Professional headshot recommended

### 2. Project Images
Replace the placeholder project images:
- `project1.jpg` - Your first project image
- `project2.jpg` - Your second project image  
- `project3.jpg` - Your third project image

### 3. Personal Information
Update the following in `index.html`:

**Contact Information:**
- Email: `andrew@example.com`
- Phone: `+1 (234) 567-890`
- Location: `San Francisco, CA`

**Social Media Links:**
- LinkedIn, GitHub, Twitter links in the contact section

**About Section:**
- Update the bio text to reflect your background
- Modify the skills list to match your expertise

**Projects:**
- Update project titles, descriptions, and technologies used
- Add your actual project links (Live Demo and GitHub)

### 4. Styling Customization
The website uses a blue color scheme (`#3b82f6`). To change colors:
- Primary blue: `#3b82f6`
- Text colors: `#1f2937` (dark), `#6b7280` (gray)
- Background: `#f8fafc` (light gray)

### 5. Adding More Projects
To add more projects, copy the project card structure in the projects section and update the content.

## Getting Started

1. Add your profile picture as `profile.jpg`
2. Add your project images
3. Update all personal information in `index.html`
4. Customize colors in `styles.css` if desired
5. Open `index.html` in your browser to view the website
